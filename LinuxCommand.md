# Linux 명령어 정리
**apt-get install [설치대상]**: (Advanced Package Tool) 패키지 관리

**sudo**: (Substitute User DO) 관리자 권한으로

**sudo -s**: 계속 관리자 권한으로 

**mkdir**: (Make Directory) 디렉도리 만들기

**mv**: (Move) 파일/디렉도리 이동 OR 이름 바꾸기

**make**: 범용적 컴파일 명령어

**tar**: (Tape Archiver) 파일 압축 및 해제

**wget**: (Web Get) 웹서버에서 컨텐츠 가져오기


**./configure**
**make**
**make clean**
